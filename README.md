voice-to-linkedin
A Make.com automation that turns voice memos into drafted LinkedIn posts using OpenAI Whisper and Claude.
How it works

Drop a voice memo (.m4a) into a Google Drive folder
Whisper transcribes the audio
Claude drafts a LinkedIn post in your voice
Draft lands in your inbox via Gmail

Tools used

Make.com
Google Drive
OpenAI Whisper
Anthropic Claude (claude-sonnet-4)
Gmail

Setup

Import the .json scenario into Make
Add your own API keys where indicated
Point the Google Drive trigger at your folder
Add your email to the Gmail step
